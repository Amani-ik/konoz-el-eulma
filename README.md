# 📍 كنز العلمة - Konoz El Eulma

أول دليل رقمي وتفاعلي مخصص للمناطق التجارية بمدينة العلمة (شارع دبي، شارع المواعين، إلخ).

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Frontend:** HTML5, CSS3 (Custom Design), JavaScript (ES6+).
- **Weather API:** OpenWeather integration.
- **Map:** Interactive Image Map with dynamic markers.

## 🚀 كيفية التشغيل (Setup)

1. قم بتحميل المشروع (Clone).
2. افتح ملف `index.html` في أي متصفح حديث.
3. للتعديل على البيانات، راجع قسم "تعديل المحتوى" أدناه.

## ✍️ تعديل المحتوى (Guide for Admin)

بما أن النسخة الحالية تعتمد على البيانات الثابتة (Hardcoded):

- **الأخبار:** ابحث عن مصفوفة `NEWS_DATA` في ملف `index.html`.
- **المحلات:** ابحث عن مصفوفة `SHOPS_DATA` لتعديل الأسماء أو الصور أو الإحداثيات.

## 🛡️ حالة المشروع

- النسخة الحالية: **v28 (Stable MVP)**.
- المتصفحات المدعومة: Chrome, Edge, Safari (iOS Optimized).
